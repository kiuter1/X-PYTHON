## Download
```
git clone https://github.com/kiuter1/X-PYTHON.git
```

# Installation
```
pip install -r requirements.txt
```

## TESTS
To run tests, use following command:

for Windows:
```
python task1.py
python task2.py
```

if you use poetry :
```
poetry init
poetry add
poetry run python task1.py
poetry run python task2.py
```
